Maelekezo ya Hosting Bure (GitHub Pages)

  1. Tengeneza akaunti kwenye GitHub.com
  2. Tengeneza repository mpya, jina: baraza-usalama-barabarani, type: public
  3. Pakia faili zote za tovuti hii ndani ya repository
  4. Jina la main file lazima liwe index.html
  5. Endelea kwenye Settings → Pages
  6. Chagua source: main branch, folder: /root, kisha save
  7. GitHub Pages itaunda link ya HTTPS mfano: https://username.github.io/baraza-usalama-barabarani
  8. Tumia link hiyo kwa majaribio, demo, au kuonesha ICT / eGA

⚠️ Hii ni kwa staging/testing tu, si production rasmi ya .go.tz
